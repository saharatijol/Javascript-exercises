Pangram Practice

A Pangram is a sentencec that contains every letter of the alphabet, like:

"The quick brown fox jumps over the lazy dog"

Write a function that is called isPangram, which checks to see if a given sentence contains every letter of the alphabet. Make sure ignore string casing!

isPangram('The five boxing wizards jump quickly') //true

isPangram('The five boxing wizards jump quick') //false
